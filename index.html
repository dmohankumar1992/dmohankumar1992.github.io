Need for Privacy

  1. The purpose of the Data Protection is to protect personal data
  2. serves to keep those who process the data responsible in how they handle and process the data
  3. Protect the data from the third party sites
  4. Secure the paswords from the hacker

How to be Secured ?

  1. Lightbeam - - See who tracks us online
  2. about:permissions - Set preferences for every website
  3. about:config - Play with Mozilla code
  4. about:privatebrowsing- No more cookies
  5. Webmaker Tools - Learn How the Web Works

Lightbeam

  1. Lightbeam is a Firefox add-on
  2. It enables you to see the first and third party sites you interact with on the Web
  3. It shows you the relationships between these third parties and the sites you visit

about:permissions

The Permissions Manager gives you easy access to the following

  • Store passwords - Set Firefox can store passwords
  • Share Location - Set Firefox share your location
  • Set Cookies - How long websites can set cookies
  • Open Pop-up windows - Allow or block pop-up windows
  • Maintain Offline Storage - websites can store information on your computer for use when you are offline
  • Fullscreen - Set whether or not websites can be displayed in full screen.

about:config

  • It is used to customize the firfox
  • we can add,modify and reset the preferences such as Speed up Firefox, Disable antivirus scanning, Open Javascript popups as tabs, Spell checking, etc

about:privatebrowsing

Private browsing can not save the following

  • Visited pages
  • Form and Search Bar entries
  • Passwords
  • Download List entries
  • Cookies
  • Cached Web Content and User Data
